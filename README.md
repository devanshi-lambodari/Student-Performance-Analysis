# Student-Performance-Analysis
Performed exploratory data analysis using Pandas, Seaborn, and Matplotlib to identify trends, score distributions, and correlations between demographic factors and student academic performance.

## Objective
To analyze student exam performance and identify trends using data analysis and visualization.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Dataset
Student Performance in Exams dataset from Kaggle.

## Key Insights
- Female students outperform in reading and writing, while males score higher in math.
- Parental level of education impacts student performance.
- Most students score between 60â€“70 marks in math.
- Strong correlation between reading and writing scores.

## Visualizations
Bar plots, histograms, and correlation heatmap were used to understand patterns in the data.
